# NextJS Project - Setup Guide

## Getting Started

1. **Clone the GitHub repository:**

   ```bash
   git clone https://github.com/omohemma/withrotate.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd withrotate
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Run NextJS development server:**

    ```bash
   npm run dev
    ```
